# **This is a simple Pokédex application built with Python.**

## **Installation guide**
- Click the Code button above and select Download ZIP.
- Extract the ZIP folder to your desired location.
- Open the extracted folder and drag the project subfolder into your preferred IDE.
- Alternatively, open the same subfolder directly from within your IDE.

## **Program Requirements**
To run the program, you will need the *following dependencies installed*:
- Python version **3.13 or higher**
- CustomTkinter version **5.2.0 or higher**
- Pandas version **2.3.3 or higher**
- Matplotlib version **3.10.8 or higher**

## **Installing and Updating Dependencies**

### **Installation**
If you do not already have Python installed, you can *download the latest version from*:
https://www.python.org/downloads

Once Python is installed, you can *install the required libraries* by running the following commands in Command Prompt:
- pip install pandas matplotlib
- pip install customtkinter

If you receive an error saying that the pip command is not recognised, use these commands instead:
- py -m pip install pandas matplotlib
- py -m pip install customtkinter

### **Updating Dependencies**
If you already have the required libraries installed, you can check whether they are up to date and meet the required versions.

### **Checking Versions**
To check the versions of the dependencies if you think you have them installed, run the following in Command Prompt:
- pip show pandas matplotlib
- pip show customtkinter

If the pip command is not recognised, use these commands instead:
- py -m pip show pandas matplotlib
- py -m pip show customtkinter

### **Updating Dependencies**
To update the dependencies if you already have them installed, run the following in Command Prompt:
- pip install --upgrade pandas matplotlib
- pip install --upgrade customtkinter

If the pip command is not recognised, use these commands instead:
- py -m pip install --upgrade pandas matplotlib
- py -m pip install --upgrade customtkinter

## **Running the Program**
After installing the dependencies, the application can be run directly through VS Code or any other Python-compatible IDE.
